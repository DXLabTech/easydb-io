# easydb-io

Código utilizado na demo de apresentação sobre easydb

## O que é easydb-io?

Easydb é um serviço de banco de dados online (database as a service). Nós indicamos ele para propositos educacionais.
Demonstrações de interações com banco de dados sem complexidades de instalações e configurações.
